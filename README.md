# API Joke Generator

A simple Python program that fetches a random joke from the public [official-joke-api](https://official-joke-api.appspot.com/random_joke).

Prints a joke with its `setup` and `punchline` directly in the terminal.

---

## Features

- Fetches a random joke with a single HTTP request
- Cleanly displays the setup and punchline
- Lightweight, only depends on `requests`

---

## Requirements

- Python 3.6 or higher
- `pip` (Python package manager)
- requests library
